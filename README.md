# 📱 Mobile Sales Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive **Mobile Sales Dashboard** built using **Power BI**. It helps analyze sales performance, customer trends, and product insights for a mobile retail business.

The dashboard provides a clear and visual representation of key metrics to support data-driven decision-making.

---

## 📊 Dataset Information

The dataset includes sales-related data such as:

* Order Date
* Product (Mobile Models)
* Sales Amount
* Quantity Sold
* Profit
* Customer/Region details (if applicable)

---

## 🎯 Objective

The main goals of this project are:

* Analyze overall mobile sales performance
* Identify top-selling products
* Track monthly/quarterly trends
* Understand profit distribution
* Provide insights for business growth

---

## 📈 Key Insights

* 📊 Top-performing mobile brands/models identified
* 📅 Sales trends observed across months/quarters
* 💰 Profit vs Sales comparison for better decision-making
* 🌍 Region-wise performance analysis (if included)
* 🛒 Quantity vs Revenue relationship

---

## 🛠 Tools & Technologies Used

* **Power BI**
* **Power Query (Data Cleaning & Transformation)**
* **DAX (Data Analysis Expressions)**

---

## 📌 Features of Dashboard

* Interactive slicers (Date, Product, Region)
* KPI cards (Total Sales, Profit, Quantity)
* Bar charts & line charts for trend analysis
* Matrix/Table view for detailed data
* Dynamic filtering and drill-down options

---

## 📂 Project Files

## 🚀 How to Use

1. Download the `.pbit` file
2. Open in **Power BI Desktop**
3. Load dataset (if prompted)
4. Explore dashboard using filters and visuals
